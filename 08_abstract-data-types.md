# Abstract Data Types

ADTs are tackled in the lecture but not in the laboratory. Four ADTs are usually discussed in CMSC12: stacks, queues, trees, and graphs.

### Stacks

1. [Delimiter Soup | Kattis](https://open.kattis.com/problems/delimitersoup) (or any bracket matching problem)

2. [Pairing Socks | Kattis](https://open.kattis.com/problems/pairingsocks)

3. [I Can Guess the Data Structure! | Kattis](https://open.kattis.com/problems/guessthedatastructure) (stack/queue/priority queue)

4. [Implement Queue using Stacks](https://leetcode.com/problems/implement-queue-using-stacks/description/)
